## The new concept: a live timer

Until now our screens reacted to *data* changes. A countdown is different — it
changes on a **clock tick**, once per second, on its own. We use a `Timer` that
fires every second and updates the remaining time. Two important details:

- We **cancel the timer** when leaving the screen (so it doesn't keep running in
  the background and waste resources). This happens in `dispose()`.
- We **save the session exactly once** (a guard flag prevents double-saving if
  the timer and the user both try to end it).

This is explained in the code comments of
`lib/presentation/focus/focus_session_screen.dart`.

## How it behaves

1. Open the **Focus** tab → a card invites you to start a session.
2. Type an intention (e.g. "Write the report"), pick **25 min**, tap **Start**.
3. A full-screen timer counts down with a progress ring.
4. Let it finish → it saves as completed and shows **Done!**; or tap **End
   session early** → it saves the minutes you managed.
5. Back on Focus home, today's stats and the session list update instantly.
